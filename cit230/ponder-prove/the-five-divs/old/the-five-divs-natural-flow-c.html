This is an example of a floated image. An image element is placed above several paragraphs of text. The image has been given an alt attribute to explain what it is for accessibility, and width and height attributes to provide a placeholder space as the image loads (or if it is not available for some reason). The image is then floated using the line float: left; in the CSS.

In case you are wondering, the image is a very strange sign I saw while on holiday with my family in Australia. It is warning passersby about a cow being electrocuted as it runs a car over, or something...?!?!
