Brackets Editor

Brackets is one of many text editors that can be used for Frontend Web Development. Brackets is recommended for our class because:

  1. It's free!
  2. It is very easy to use
  3. It works on Linux, Mac and Windows
  4. It looks and behaves basically the same on Linux, Mac and Windows
  5. It has an active development cycle which means it is being updated regularly and new add-on tools are being made available regularly

Recommended Brackets Extensions

  • Autoprefixer — Parse CSS and add vendor prefixes automatically.
  • Beautify — Format JavaScript, HTML, and CSS files
  • Brackets Icons — add file icons to the file tree to more easily identify file types
  • Indent Guides — Show indent guides in the code editor.
  • Join Lines — Joins the next line or all the lines selected, with or without whitespace.
  • JS CSS Minifier — Minifies your JavaScript and CSS files with UglifyJS2 and CleanCSS.
  • QuickdocsJS — Inline documentation for JavaScript functions, including a summary, syntax and parameters. 
  • QuickdocsRegEx — Inline documentation for regular expressions in JS and PHP.
  • Paste and Indent — automatically apply the correct indentation to pasted text
  • White Space Sanitizer — trims trailing white spaces; transforms tabs to spaces; adds newline at the end of file
